# Project Phase 3 - Cloud Dashboard Optimization

## Stack
- Backend: Python Azure Function
- Storage: Azure Blob Storage
- Cache/Data: JSON (processed data stored after computation)
- Frontend: HTML, CSS, JavaScript, Chart.js
- Authentication: Firebase Auth (Email/Password + OAuth)
- Hosting: Azure Static Web App

## Team Split

### Person 1 – Backend & Performance
- Blob trigger function
- Data cleaning and processing
- API optimization using cached data

### Person 2 – Authentication & Security
- User login system
- OAuth integration
- Secure password storage

### Person 3 – Frontend & Interaction
- Dashboard UI updates
- Filters, search, pagination
- User session handling

---

## Key Features
- Fast API responses using cached data
- Secure login system
- Interactive dashboard with filters and search
- Scalable cloud-based architecture
